关于 SBTI 毒舌人格测试的常见疑问

什么是SBTI人格测试？和MBTI有什么区别？

SBTI测试（Soul Behavior Type Indicator）是风靡全网的新一代免费人格测试工具，由B站UP主蛆肉儿串儿原创，后经AI二创优化。与传统MBTI测试不同，SBTI基于15个更深度的细分维度（自尊自信、依恋安全感、世界观倾向、执行模式等），放弃了MBTI那种"听起来全是优点"的刻板4字母标签体系，直击真实人格的阴暗面和亮点。测试内置DeepSeek AI毒舌锐评功能，被许多用户评价"比MBTI准多了，扎心但我认"。

SBTI测试完全免费吗？需要下载App或注册吗？

SBTI人格测试完全免费，无需下载，无需注册账号，打开网页即可直接测试。整个测试约3分钟完成，结果即时生成，AI毒舌锐评也是免费的。本站服务器和AI算力均为站长自费维护，欢迎传播分享，切勿滥用API接口。

SBTI有哪些人格类型？哪些人格最稀有？

SBTI测试包含25种以上人格类型，分正常人格池和隐藏人格池两层。常见类型包括：CTRL拿捏者、GOGO行人、BOSS领导者、THAN-K感恩者、MUM妈妈、FAKE伪人、OJBK无所谓人、MALO吗喽、SHIT愤世者、DEAD死者、ZZZZ装死者等。最稀有的是CTRL和BOSS（系统标注仅占极少数人口）。隐藏人格DRUNK酒鬼需在特定题目中选择"饮酒"爱好才能触发，属于埋藏的彩蛋人格。

SBTI隐藏人格怎么解锁？酒鬼人格如何触发？

SBTI测试中有隐藏的酒鬼人格（DRUNK）可以解锁。当测试中出现"您平时有什么爱好？"这道特殊题目时，选择"饮酒"选项，随后系统会追加一道关于饮酒态度的题目，根据整体测试结果决定是否触发DRUNK人格。此外，HHHH傻乐者是保底人格——当答题结果无法匹配任何正常人格时自动触发，属于作者"故意埋的损招"。

SBTI测试结果准不准？可以相信吗？

SBTI测试定位为趣味性格娱乐测试，基于原创15维度量表设计，许多参与者反映结果"扎心但很准"，尤其是15维度的细分解读让人直呼"被看穿了"。不过请注意，SBTI与MBTI一样，均不具备临床心理学或医学诊断价值，结果仅供参考和娱乐。如有心理健康相关需求，请寻求专业心理咨询师帮助。

SBTI的AI毒舌锐评是怎么工作的？

测试完成后，点击"AI毒舌锐评此性格"按钮即可触发。系统会将你的15维度评分数据提交给DeepSeek大模型，AI深度推理后生成一段量身定制、辛辣毒舌但真诚的性格点评，以打字机效果逐字呈现。每个人报告均不同，正是"千人千面"的设计理念，你永远不会看到和朋友完全相同的AI分析。

SBTI的题目是什么风格？会不会太无聊？

SBTI题目风格幽默、直白、甚至有些毒舌，与严肃心理量表截然不同。题目会描述让人哭笑不得的荒诞场景，比如"便秘坐马桶30分钟你会怎么办"、"萌萌小女孩给你棒棒糖你怎么看"。这种设计降低了心理负担，同时更能探测出真实的行为反应模式，而非经过包装的"理想答案"，是许多用户认为SBTI比MBTI"更真实"的原因之一。

SBTI的测试进度会保存吗？关掉页面再回来还有吗？

SBTI测试会自动保存做题进度。关闭浏览器再打开，首页会显示"继续上次未完成的测试"按钮，点击即可从断点处继续作答，无需从头重来。如果想全新测试，点击"从头重新测试"按钮即可清除进度重新开始。
