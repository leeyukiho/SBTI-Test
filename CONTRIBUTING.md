# Contributing to SBTI Personality Test

感谢你对 SBTI 人格测试项目的关注！我们欢迎所有形式的贡献，包括修复 BUG、改进文档或提出新功能建议。

## 如何贡献

### 提交 Issue
如果你发现了问题，请先在 [Issues](https://github.com/leeyukiho/SBTI-Test/issues) 中搜索是否已有类似反馈。如果没有，请创建一个新 Issue，并详细描述：
- 问题的具体表现
- 复现步骤
- 期望的行为与实际行为的差异

### 提交 Pull Request
1. Fork 本仓库。
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)。
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)。
4. 推送到分支 (`git push origin feature/AmazingFeature`)。
5. 开启一个 Pull Request。

## 开发规范
- **保持原生**：本项目坚持使用 Vanilla JS (原生 JS)，请避免引入 React/Vue/Tailwind 等大型框架或库。
- **SEO 优先**：任何页面大幅改动都应考虑对 SEO 的影响，保持语义化 HTML。
- **移动端适配**：所有 UI 改动必须经过移动端浏览器兼容性测试。

## 行为准则
请保持友善与尊重。我们致力于建立一个多元且包容的开源社区。
